Installation
=============

There are multiple ways to install SolarThing.
Using Docker has been supported since version 2023.3.0 and is the recommended way to install and use SolarThing.

.. tabs::

  .. group-tab:: Docker Install

    To install SolarThing on your system using Docker, go to :doc:`docker`.

  .. group-tab:: Native Install

    To install SolarThing natively on your system, go to :doc:`native`.
